1. Create the project "test"

   ```powershell
   idf.py create-project test
   ```

2. Enter the "test" folder

3. Set board to "ESP32S3"

   ```powershell
   idf.py set-target esp32s3
   ```

4. Build 

   ```powershell
   idf.py build
   ```

5. Flash to board on COM5

   ```powershell
   idf.py -p COM5 flash
   ```

# Partition Table

0x0000

​	Boot

​	Comes with the system

​	启动区域，类似电脑BIOS

0x8000

​	Partition table

​	Comes with the system

​	分区表烧入区域，可使用分区表修改后面的数据

0x9000

​	NVS

​	非易失存储区域

0xf000

​	phy_init

​	PHY数据存储，主要是关于无线（WiFi）数据的存储

0x10000

​	factory

​	Comes with the system

​	程序数据存储，包括系统自带的和自己编写的

0x200000

​	vfs

​	虚拟文件系统数据存储区域

0xc00000

​	storage

​	用户自定义区域，可个性化修改此区域

0x1000000
